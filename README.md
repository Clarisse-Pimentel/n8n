# Projeto n8n - Tratamento de Erros e Subworkflows

Projeto desenvolvido utilizando n8n com foco em:

* Tratamento de erros
* Retry de APIs
* Modularização
* Subworkflows
* Backup/versionamento

Funcionalidades implementadas:

* HTTP Request
* IF para validação
* Set para mensagens de erro
* Execute Subworkflow
* Retry On Fail
